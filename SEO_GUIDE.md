# SEO & Google Indexing Guide for ApplyBoost

## Files Added for SEO

✅ **robots.txt** - Tells search engines which pages to crawl
✅ **sitemap.xml** - Lists all pages for search engines to index
✅ **Meta tags** - Added to index.html for better search visibility

## Step 1: Submit to Google Search Console

### A. Verify Your Site

1. Go to **https://search.google.com/search-console**
2. Click "Add Property"
3. Enter: `applyboost.ai`
4. Choose verification method:

**Option 1: HTML File Upload (Easiest)**
- Download the verification file (e.g., `google1234567890.html`)
- Add it to your repository root
- Commit and push to GitHub
- Click "Verify" in Search Console

**Option 2: DNS Verification**
- Copy the TXT record provided
- Add to Namecheap DNS settings
- Wait 5-10 minutes
- Click "Verify"

**Option 3: HTML Meta Tag**
- I can add the meta tag to index.html
- Just provide the verification tag

### B. Submit Your Sitemap

1. In Google Search Console, go to **Sitemaps** (left menu)
2. Enter: `https://applyboost.ai/sitemap.xml`
3. Click "Submit"
4. Google will start crawling your site within 24-48 hours

### C. Request Indexing

1. In Search Console, go to **URL Inspection**
2. Enter: `https://applyboost.ai`
3. Click "Request Indexing"
4. Repeat for important pages:
   - `https://applyboost.ai/privacy.html`
   - `https://applyboost.ai/terms.html`
   - `https://applyboost.ai/gdpr.html`

## Step 2: Submit to Other Search Engines

### Bing Webmaster Tools

3. Verify ownership (similar to Google)
4. Submit sitemap: `https://applyboost.ai/sitemap.xml`
